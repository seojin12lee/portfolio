About

저는 서비스 구조를 이해한 상태에서 보안을 설계·점검할 수 있는 보안 엔지니어를 지향합니다. 단순 취약점 도구 사용에 그치지 않고, 웹·시스템·네트워크를 직접 구축한 뒤 공격 시나리오를 설정하고 로그로 검증하는 방식의 실습을 반복해왔습니다.

특히 DNS, 로그 서버, 웹 서버와 같은 실제 운영 인프라 구성 요소를 직접 다뤄보며 사고 발생 지점과 분석 근거를 중심으로 보안 대응 경험을 쌓았습니다.
